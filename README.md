This is a short project analyzing changes in NFL decision making on 4th down. Comparisons are made between the 2000 and 2025 NFL season using complete 
play-by-play datasets from these seasons. This data was obtained from the nflverse GitHub, and can be found here: 
https://github.com/nflverse/nflverse-data/releases/tag/pbp

The project makes use of SQL and Pandas to analyze and visualize data.
